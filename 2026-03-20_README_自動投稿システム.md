## 1. システム概要

Xのアルゴリズム（Author Diversity Scorer / AgeFilter / score_recent）の分析結果にもとづき、  
**朝7時・昼12時の1日2回**、自動的に投稿とSlack通知を行う仕組みです。

```
[Macのスケジューラー（launchd）]
        ↓ 毎日 07:00 / 12:00 に起動
[imakentei_post_once.py]
        ↓ post_data.json を確認して投稿
[X API（Tweepy）]
        ↓ 投稿完了 or エラー
[Slack Incoming Webhook]
        ↓ 通知を送信
[あなたのSlack]
```

## 4. 環境設定（.env）

`.env` ファイルに以下のキーを設定しています。**このファイルは絶対に外部に共有しないこと。**

```
X_API_KEY=…         # X Developer Portal のAPI Key
X_API_SECRET=…      # X Developer Portal のAPI Key Secret
X_ACCESS_TOKEN=…    # アクセストークン
X_ACCESS_SECRET=…   # アクセストークンシークレット
SLACK_WEBHOOK_URL=… # Slack Incoming Webhook URL
```

---

## 5. 自動起動の仕組み（macOS launchd）

macOSのビルトインスケジューラー **launchd** を使用しています。  
設定ファイルは以下のパスに保存されています：

```
~/Library/LaunchAgents/com.imakentei.autopost.plist
```

**動作タイミング：毎日 07:00 と 12:00 の2回**  
Macにログインしていれば、PCが起動するたびに自動的に有効になります。

## 6. よくある操作

### ▶️ 自動投稿を一時停止したい
```bash
launchctl unload ~/Library/LaunchAgents/com.imakentei.autopost.plist
```

### ▶️ 自動投稿を再開したい
```bash
launchctl load ~/Library/LaunchAgents/com.imakentei.autopost.plist
```

### ▶️ 手動でテスト投稿したい（今すぐ1件確認）
`post_data.json` の対象の `date` と `time` を現在時刻に変更してから以下を実行：

python3 imakentei_post_once.py

コンテンツ案のMarkdownを更新した後に実行：

## 7. ⚠️ 運用上の注意点（アルゴリズム対策）

| 注意事項 | 理由 |
|---|---|
| **リプライには24時間以内に必ず返信する** | Author Reply は +75.0 の最高スコア行動 |
| **1時間以内の手動追加投稿は厳禁** | Author Diversity Scorer でスコアが減衰する |
| **Macをスリープ・シャットダウンしない（7時・12時前後）** | launchd はスリープ中は実行されない |
| **APIプランを確認する（画像付き投稿）** | 画像アップロードはBasicプラン以上が必要な場合あり |
